# User Records

## Description

A simple PHP and MySQL project to manage user records.

## Features

- Add new users
- Display all users
- Toggle user status
- Store data in MySQL database
- Uses Fetch API without page refresh

## Technologies Used

- HTML
- JavaScript
- PHP
- MySQL

## Database

Import the `user.sql` file into MySQL before running the project.
